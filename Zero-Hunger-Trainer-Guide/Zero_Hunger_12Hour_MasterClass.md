### What is JavaScript?
JavaScript is a programming language used to create interactive web applications.

**Without JavaScript:**
- Website = Static
- No Validation
- No Dynamic Data
- No Interactivity

**With JavaScript:**
- Interactive Forms
- Live Search
- Dashboards
- API Calls
- Animations

### Real Life Example
Imagine Amazon.  
When you click **Add To Cart**, the page updates instantly.  
*JavaScript is doing that.*

### Zero Hunger Example
